## Rubies

__Lotus::Validations__ supports Ruby (MRI) 2+ and JRuby 1.7 (with 2.0 mode).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'lotus-validations'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install lotus-validations
